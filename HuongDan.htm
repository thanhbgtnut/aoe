I - HƯỚNG DẪN CƠ BẢN

Chạy file SETUP.EXE ở lần đầu chơi game, khi chuyển game qua thư mục khác để cài đặt và sửa một số lỗi thường gặp.

- Đăng ký tất cả các khoá registry cần thiết

- Tạo đường dẫn trên desktop và Start Menu

- Cài đặt DirectPlay để chơi AoE qua mạng, khắc phục hiện tượng không hiện mạng Internet TCP/IP Connection For DirectPlay trong Multiplayer.

- Cho phép AoE truy xuất qua Firewall (Tường lửa) để chơi qua mạng.

- Tắt UPnP an toàn, khắc phục hiện tượng show/create không thấy phòng khi chơi online.

- Fix lỗi nhoè màu trên Windows 7; lỗi game chỉ hiển thị một góc màn hình.

- Chọn độ phân giải cho game.

- Lựa chọn tuỳ chọn chế độ cửa sổ và tiếng Việt cho game.

1. Thay đổi độ phân giải trong game:

Nhấn phím 1 và lựa chọn Set độ phân giải phù hợp. Từ độ phân giải cho màn VGA 4:3 cũ (1024x768, 800x600x 640x480), được nâng cao phù hợp với các màn hình hiện nay. Mỗi file chỉ hỗ trợ 3 độ phân giải nên chia ra 4 file exe, mỗi file hỗ trợ 1 set gồm:

1) Độ phân giải mặc định (màn hình 16:9, phổ biến nhất):

1920 x 1080  1600 x 900  1366 x 768

Màn 2K cũng dùng được set này.

2) Set 2K: Cho màn 16:9 2K và màn khổ lớn (màn hình 16:9 hoặc 21:9):

            2560x1440  1920x1080  1600x900

Ngoài ra còn phù hợp với màn 16:9 khổ lớn hơn như 2880x1620, 3200x1800, 3520x1980, 3840x2160 (4K), 4096x2304 (Full 4K), và các màn khổ lớn tỷ lệ 2:1, 2.39:1, 2.75:1.

3) Set 16:10: Hỗ trợ màn hình tỉ lệ 16:10:

            1680x1050  1440x900  1280x800

Ngoài ra còn phù hợp với màn 1920x1200 và 2560x1600.

4) Set hỗ trợ độ phân giải khác:

            1360x768  1280x1024  1024x768

Ngoài ra phù hợp với màn 4:3 lớn hơn như 1400x1050, 1440x1080, 1600x1200.

● AoE HD sẽ hiển thị tốt nhất nếu dùng đúng một trong 4 set trên, sao cho đúng với thông số độ phân giải màn hình, hoặc đúng với tỷ lệ khung hình. Nếu không game sẽ báo lỗi không chạy được, hoặc hình ảnh bị méo, nhoè.

Để kiểm tra màn hình mình độ phân giải bao nhiêu, nhấn chuột phải màn hình chính Desktop và chọn:

- Windows XP: Chọn Properties > Settings.

- Windows 7, 8, 8.1: Chọn Screen Resolution

- Windows 10: Chọn Display Settings, nhìn vào Resolution, hoặc chọn Advanced display settings.

● Khi đang chơi game, chọn Menu (F10), Game Settings. Trong phần Screen Size chọn một trong 3 độ phân giải phù hợp.

● Nếu bạn thấy hình ảnh nhỏ, hãy chỉnh về độ phân giải thấp hơn. Trong quá trình chơi có thể thay đổi độ phân giải, thuận tiện cho việc lên đời hay điều quân. VD: Bạn dùng màn 1920x1080, giai đoạn lên đời có thể chọn độ phân giải 1600x900 để dễ thao tác câu voi, lùa hươu; lúc phát triển to, điều quân chỉnh lên 1920x1080 để bao quát được bản đồ.

2. Tuỳ chọn Chế độ cửa sổ:

Chạy file Setup.exe, chọn 2, chọn tiếp:

1: Cài đặt chế độ cửa sổ.

2: Trở về chế độ toàn màn hình ban đầu.

- Lưu ý: Muốn game hiện trong cửa sổ, độ phân giải của game phải NHỎ HƠN độ phân giải của màn hình bạn.

VD: Nếu màn hình của bạn là 1920x1080, bạn cần cài độ phân giải game là 1600x900 mới có cửa số.

Với màn 1366x768 (thường có ở laptop) cần patch set 1024x768 trước, tuy nhiên sẽ không hiện thanh Title Bar.

- Cách di chuyển cửa sổ: Nhấn phím Window, đưa chuột lên nhấn giữ thanh Title Bar kéo đi.

Chế độ cửa sổ giúp game hiện đúng với độ phân giải hiện tại của game, dễ dàng chuyển qua tác vụ khác (Alt+Tab) nhanh chóng so với chế độ toàn màn hình mặc định.

3. Tuỳ chọn tiếng Việt cho game

Tiếng Việt của game là tiếng Việt chuẩn chính thức, dịch bởi Microsoft, lấy từ bản AoE DE. Tên các nền văn minh được Việt hoá: Egypt - Ai Cập, Greek - Hy Lạp, Persia - Ba Tư, Roma - La Mã.

4. Cài đặt và sửa lỗi cơ bản

Setup.exe thực hiện các công việc sau (phím 9 nếu muốn cài lại):

- Đăng ký tất cả các khoá registry cần thiết.

- Cài và thiết đặt DirectDraw, fix lỗi nhoè màu trên Win 7, khắc phục lỗi ảnh chụp màn hình màu đen, lỗi game chỉ hiển thị một góc màn hình.

- Tạo đường dẫn trên desktop có icon độ nét cao dẫn đến file empiresx; trên Start Menu dẫn tới empires, empiresx và thư mục ảnh chụp màn hình.

- Cài đặt DirectPlay để chơi AoE qua mạng, khắc phục hiện tượng không hiện mạng Internet TCP/IP Connection For DirectPlay trong Multiplayer.

- Cho phép AoE truy xuất qua Firewall (Tường lửa) để chơi qua mạng mà không cần phải tắt tường lửa.

- Tắt UPnP an toàn, khắc phục hiện tượng show/create không thấy phòng khi chơi online.

- Cài font mặc định của game (Copperplate Gothic). Cài các font tiếng Việt cần thiết.

4. Chơi game bằng một trong các cách sau:

- File EMPIRESX.EXE hoặc empires.exe trong thư mục AoE-HD.

- Biểu tượng Age of Empires HD trên màn hình chính Desktop (dẫn đến file EMPIRESX.EXE).

- Trong Start Menu/Age of Empires HD.

- Chơi bằng Garena (Xem ở mục II).

● Hai file chạy của game gồm:

* File EMPIRESX.EXE: Lấy từ bản AoE chuẩn thi đấu của Việt Nam và TQ, có chức năng hiện tên phòng game và lưu cài đặt bản đồ khi chơi Multiplayer, nhưng không gõ lệnh được. Ngoài ra được hỗ trợ chống các loại hack data, tài nguyên, dân quân, ai dùng hack sẽ bị dis, nổ nhà, chết dân, defeat và hiện thông báo trên thanh chat: "Player '...' has been kicked: ... hack detected". Có 4 file hỗ trợ 4 set độ phân giải đã kể ở trên.

* File empires.exe (icon có nền đen): Lấy từ file empiresx.exe của Microsoft, chơi Single Player có thể gõ lệnh, dân số tối đa 1000, hỗ trợ set 16:9 mặc định. Lưu ý đây là file Đế chế xanh đã đổi tên, còn file Đế chế đỏ đã được xoá bớt.

5. Những thay đổi khác của AoE HD:

Game chủ yếu thay đổi về mặt hiển thị. Tất cả cân bằng game giữ nguyên để có thể tương thích và chơi song song với bản 1.0 thường, đây là điều quan trọng nhất.

● Tương thích với các bản Win và fix các lỗi lag, dis.

● Cấu trúc thư mục AoE được tinh gọn, ảnh chụp màn hình (Ctrl + F12) được lưu trong thư mục Screenshot, khắc phục hiện tượng file ảnh AoE*.bmp và file game?.nfo lưu tràn lan đầy thư mục.

● Có 50 giọng đọc (taunt 1-50) như bản AoE thi đấu.

● Thêm 6 bài nhạc Đế chế của Microsoft bị ẩn (thành 16 bài), file nhạc có tên là music1.mid, music2.mid... music16.mid. Mặc định game để volume Music là 0. Để tăng volume, chọn Menu, Game Settings, kéo thanh Music lên.

● Thêm 3 chiến dịch (Campaign) còn thiếu do Microsoft tạo ra là Armies at War, a Combat Showcase; Reign of the Hittites và First Punic War.

● Chất lượng hiển thị tốt hơn.

- Thanh HP của các unit, building sẽ hiển thị đẹp hơn, không quá cao hay quá thấp (VD: thanh HP của Lạc đà quá thấp).

- Một số lỗi hình ảnh được fix lại (như ngọn lửa chòi canh hiện sai vị trí).

- Màu sắc trên minimap dễ phân biệt hơn giữa các màu quân cũng như địa hình. VD màu biển nhạt hơn dễ phân biệt với màu 1.

● Cải thiện Computer (máy) khó hơn. Máy sẽ giảm thiểu việc mua các unit như rìu, quẩy đá, chòi, thành, tập trung mua các unit tuỳ loại quân và sản quân liên tục số lượng lớn chứ không đẻ vài chục quân xong xây Wonder như cũ. Bạn có thể xây dựng cách đánh cho Máy bằng cách sửa các file *.ai trong thư mục data2 dùng Notepad.

● Phần chơi đơn (Single Player): Tăng dân số tối đa (pop limit) từ 50 lên 250 và 1000.
